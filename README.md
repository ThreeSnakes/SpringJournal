`실전 스프링 부트 워크북`으로 `SpringBoot`를 공부하면서 책에나온 예제를 따라서 실습해본 `repository`이다.

책에 나온 코드 위주로 실습 하였으며, 일부 코드는 변경하기도 하였다.

## 사용 버전
* Java 8
* Spring Boot 2.0.1

## Dependancy
- Templet
    * thymeleaf
- SQL
    * JPA
    * H2
- WEB

## 

## 단순 실행방법
* ./gradlew bootRun
* ./gradlew bootRun -Dspring.profiles.active=[프로필명]
